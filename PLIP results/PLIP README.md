# PLIP Results

This folder contains the protein–ligand interaction analyses generated using the Protein–Ligand Interaction Profiler (PLIP).

The deposited files document the noncovalent interactions identified between modeled *Trichuris trichiura* β-tubulin receptors and docked benzimidazole ligands. PLIP was used to characterize interactions such as hydrogen bonds, hydrophobic contacts, salt bridges, π–π stacking, π–cation interactions, and other protein–ligand contacts.


The PLIP results are organized according to receptor model and ligand.

Example:

PLIP results/
├── WT/
│   ├── ALBZ/
│   ├── MBZ/
│   └── OFZ/
├── G308W/
├── A352D/
├── F167Y/
├── E198A/
└── F200Y/
