# Protein–Ligand Complexes

This folder contains the selected docked ligand poses and the corresponding receptor–ligand complex structures used for PLIP analysis and molecular visualization.

The complexes were generated by combining the selected docked ligand pose with the appropriate β-tubulin receptor structure.

## Folder organization

Files are organized according to receptor model and ligand.

Example:

```text
complex/
├── WT/
│   ├── ALBZ/
│   ├── MBZ/
│   └── OFZ/
├── G308W/
├── A352D/
├── F167Y/
├── E198A/
└── F200Y/
