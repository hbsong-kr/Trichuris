# Trichuris
Investigation on Trichuris Trichiura infection (whipworm, one of the soil-transmitted helminthiasis) 

This repository contains the structural bioinformatics data, protein models, molecular docking outputs, interaction analyses, and binding-pocket tunnel results generated for the β-tubulin component of our study on Korean Trichuris trichiura isolates.
The deposited files support the comparison of the wild-type β-tubulin protein with two Korean-specific amino acid variants, G308W and A352D. The workflow includes homology modeling using SWISS-MODEL, mutant construction in PyMOL, structural quality assessment using PROCHECK and Verify3D, ligand preparation, molecular docking using AutoDock Vina, independent validation using GNINA, protein–ligand interaction profiling using PLIP, and tunnel analysis using MOLEonline.
Three benzimidazole compounds were evaluated: albendazole oxide, mebendazole, and oxfendazole. Each receptor–ligand combination was subjected to repeated docking runs using a grid centered on the canonical benzimidazole-binding region associated with residues F167, E198, and F200. Docking robustness was further evaluated using an expanded search grid and CNN-based rescoring in GNINA.
The repository includes protein sequences, wild-type and mutant structural models, prepared receptor and ligand files, docking configuration files, docking poses, scoring outputs, PLIP interaction reports, PyMOL visualization files, model-quality reports, tunnel-analysis outputs, and consolidated summary tables. Raw analytical outputs are retained alongside processed datasets to facilitate transparency, verification, and reproducibility.
The files are organized according to the major analytical stages of the study, including model preparation, quality control, docking, interaction analysis, and tunnel analysis. Descriptive filenames identify the receptor, ligand, docking run, and selected pose to allow direct traceability between raw outputs, summarized results, and the associated manuscript.
This repository is intended to serve as the supporting computational dataset for the associated publication and to enable independent examination and reproduction of the structural and molecular docking analyses.
